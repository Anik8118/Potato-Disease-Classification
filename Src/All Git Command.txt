***Clone the GitHub repository
git clone https://github.com/Md-EmranHossen/Potato-Disease-Classification.git

***Navigate into the cloned repository directory
cd Potato-Disease-Classification

***Pull the latest changes from the remote repository
git pull origin main

***Check the status of your working directory
git status

***Add all changes to the staging area
git add .

***Commit the changes with a meaningful commit message.
git commit -m "Your commit message"

***Push the changes to the remote repository on the 'main' branch
git push origin main

***For future updates, to pull the latest changes and avoid conflicts
git pull origin main
